Design rules

  • One token set (:root) drives Word + Excel — both add-ins link design-system.css.
  • Density = 0 (Fluent default). 4 px spacing grid. 28 px control min-height.
  • Text on dark surfaces stays AA against the chosen neutral ink ramp.
  • Proposal card is a shared component; both add-ins render the same JSX.
  • Status colors reserved for pending / applied / error — never for chrome.
  • Reduced motion: @media (prefers-reduced-motion: reduce) disables the slide-in animation.
